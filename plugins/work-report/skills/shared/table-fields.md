# 报告表格字段规范

## 日报表格 (12字段)

| 字段 | 说明 |
|------|------|
| 序号 | 任务编号 |
| 类型 | 任务类型（页面/操作/系统/模块等） |
| 任务名称 | 任务描述 |
| 接收任务时间 | 任务接收日期 |
| 任务规定完成时间 | 预计完成日期 |
| 预计任务耗时 | 预估工时（小时） |
| 当前进度 | 完成百分比 |
| 实际交付时间 | 实际交付日期 |
| 当天耗时 | 当日工时（小时） |
| 总耗时 | 累计工时（小时） |
| 是否存在异常 | 异常情况说明 |
| 其中当天AI处理的百分比 | AI辅助百分比 |

---

## 周报/月报表格 (9字段)

| 字段 | 说明 |
|------|------|
| 序号 | 任务编号 |
| 类型 | 任务分类 |
| 名称 | 任务名称 |
| 接收任务时间 | 接收日期 |
| 规定交付时间 | 要求交付日期 |
| 实际交付时间 | 实际交付日期 |
| 开发耗时(小时) / 修复耗时(小时) | 总工时（小时） |
| 状态 | 完成/进行中 |
| 备注 | 额外说明 |

---

## 日报到周报/月报字段映射

```javascript
// 日报12字段 -> 周报/月报9字段
日报字段: 序号, 类型, 任务名称, 接收任务时间, 任务规定完成时间,
         预计任务耗时, 当前进度, 实际交付时间, 当天耗时,
         总耗时, 是否存在异常, AI处理百分比

周报/月报字段: 序号, 类型, 名称, 接收任务时间, 规定交付时间,
             实际交付时间, 开发/修复耗时, 状态, 备注

映射规则:
- 名称 = 任务名称
- 规定交付时间 = 任务规定完成时间
- 开发/修复耗时 = 总耗时(汇总同一任务的总耗时)
- 状态 = 根据当前进度判断(100%=已完成, <100%=进行中)
- 备注 = 组合(是否存在异常 + AI处理百分比)
```

---

## 表格格式要求

### Markdown 表格格式
```markdown
| 序号 | 类型 | 任务名称 | 接收时间 | ... |
|------|------|----------|----------|-----|
| 1 | [操作] | 用户登录功能 | 2025-01-01 | ... |
```

### 时间格式
- 日期: `YYYY-MM-DD`
- 时间: `YYYY-MM-DD HH:MM`

### 数值格式
- 工时: 小时，支持小数（如 1.5）
- 进度: 百分比（如 80%）
- AI百分比: 百分比（如 30%）

---

## 统计信息格式

### 日报统计
```markdown
**总计工作时长**: X.X小时
```

### 周报/月报统计
```markdown
*总工时：XX小时 | AI辅助率：XX% | 完成任务：XX项*
```

### 统计计算规则

```javascript
// 总工时计算
总工时 = sum(所有任务的当天耗时)  // 从日报
总工时 = sum(所有任务的开发/修复耗时)  // 从周报/月报

// AI辅助率计算
AI辅助率 = sum(所有任务的AI百分比) / 任务总数

// 完成任务数
完成任务数 = count(进度 == 100%) 或 count(状态 == "已完成")
```
