# 周报生成 Agent Skills 使用指南

## 简介

这是一个智能工作周报生成助手,可以自动汇总本周日报或通过交互式对话来生成符合规范的工作周报。

## 核心特性

- ✅ **日报自动汇总**: 自动读取本周所有日报文件并智能汇总
- ✅ **交互式填写**: 友好的对话式引导,逐步完成周报
- ✅ **智能分类**: 自动将任务分类到新增功能、交互优化、Bug修复
- ✅ **数据统计**: 自动计算总工时、AI辅助率、完成任务数
- ✅ **数据验证**: 确保所有字段完整,逻辑正确
- ✅ **灵活模式**: 支持4种不同的周报生成方式

## 如何使用

### 方式1: 通过 Slash 命令(推荐)

```
/generate-weekly-report
```

### 方式2: 直接调用

```
请使用 weekly-report skill 帮我写本周的周报
```

## 支持的4种模式

### 1. 从日报汇总(最强大)

**适用场景**: 本周每天都填写了日报,需要汇总成周报

**功能**:
- 自动查找本周日报文件(日报_2025-10-14.md ~ 日报_2025-10-20.md)
- 自动提取所有任务并分类
- 自动计算统计数据
- 生成工作概述和关键进展

### 2. 交互式填写

**适用场景**: 手动逐步填写周报内容

**流程**: 引导填写工作概述、各类任务、关键进展等

### 3. 从数据源导入

**适用场景**: 有其他格式的工作记录需要转换

### 4. 空白模板

**适用场景**: 需要手动编辑周报

## 周报结构

### 四个主要部分

1. **本周工作总结**
   - 工作概述
   - 新增功能开发(表格)
   - 交互调整或优化(表格)
   - Bug修复(表格)

2. **关键进展**
   - 2-3个重点进展

3. **问题反馈**
   - 遇到的问题和困难

4. **下周计划**
   - 下周主要任务

### 底部统计
`*总工时：XX小时 | AI辅助率：XX% | 完成任务：XX项*`

## 智能功能

### 任务自动分类

从日报中提取的任务会自动分类到对应表格:
- 包含"优化"、"调整" → 交互调整或优化
- 包含"修复"、"Bug" → Bug修复
- 其他 → 新增功能开发

### 类型自动推断

根据任务名称自动判断类型标签:
- 新增功能: [大系统]、[小系统]、[大模块]、[小模块]、[页面]
- Bug修复: [列表页]、[详情页]、[新增]、[删除]、[编辑]等

### 数据自动统计

- 总工时 = 所有任务耗时总和
- AI辅助率 = 所有任务AI百分比的平均值  
- 完成任务数 = 状态为"已完成"的任务数

## 最佳实践

1. **每天写日报**: 使用 daily-report skill,为周报汇总提供数据
2. **周五下班前生成**: 选择"从日报汇总"模式,自动生成周报
3. **及时补充**: 添加关键进展和下周计划
4. **定期回顾**: 周报是很好的工作回顾工具
