# 工作月报生成命令

此命令调用智能月报生成 Agent Skill,提供强大的月报生成功能。

## 使用说明

### 基本用法

```
/generate-monthly-report
```

这将启动月报生成 Agent,提供以下选项:

1. **从周报汇总** - 自动读取本月所有周报文件并智能汇总生成月报(推荐)
2. **从日报汇总** - 自动读取本月所有日报文件并汇总生成月报
3. **交互式填写** - 通过对话逐步填写月报各部分内容
4. **从数据源导入** - 从指定文件或目录导入数据生成月报
5. **空白模板** - 生成基于模板的空白月报供手动填写

### 带参数用法

如果提供了数据源路径,将自动使用"从数据源导入"模式:

```
/generate-monthly-report /path/to/reports/
```

## 核心特性

- **周报自动汇总**: 查找本月周报文件并自动汇总(优先模式)
- **日报自动汇总**: 备选方案,从本月日报汇总生成月报
- **智能任务分类**: 自动分类到新增功能、交互优化、Bug修复
- **任务去重**: 自动检测并合并跨周/跨天的重复任务
- **数据自动统计**: 计算总工时、AI辅助率、完成任务数
- **符合规范**: 严格按照月报模板格式生成
