# 工作周报生成命令

此命令调用智能周报生成 Agent Skill,提供强大的周报生成功能。

## 使用说明

### 基本用法

```
/generate-weekly-report
```

这将启动周报生成 Agent,提供以下选项:

1. **从日报汇总** - 自动读取本周所有日报文件并智能汇总生成周报
2. **交互式填写** - 通过对话逐步填写周报各部分内容
3. **从数据源导入** - 从指定文件或目录导入数据生成周报
4. **空白模板** - 生成基于模板的空白周报供手动填写

### 带参数用法

如果提供了数据源路径,将自动使用"从数据源导入"模式:

```
/generate-weekly-report /path/to/daily-reports/
```

## 核心特性

- **日报自动汇总**: 查找本周日报文件并自动汇总
- **智能任务分类**: 自动分类到新增功能、交互优化、Bug修复
- **数据自动统计**: 计算总工时、AI辅助率、完成任务数
- **符合规范**: 严格按照周报模板格式生成
